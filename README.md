# 3D Educational Game Development Project
A game development project showcase for a college assignment using Unity 3D and C#.

## Controls

### PC:
| Control    | Function              |
|------------|-----------------------|
| **W**      | Walk Forwards         |
| **A**      | Strafe Left           |
| **S**      | Walk Backwards        |
| **D**      | Strafe Right          |
| **Space**  | Jump                  |
| **E**      | Interact/Continue     |

### Mobile:
Built-in controls will be visible on the screen, similar to a typical gamepad.
